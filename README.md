## CredixAI

**CredixAI** is an advanced credit scoring application that leverages a hybrid machine learning architecture combining Multi-Layer Perceptron (MLP) and XGBoost within a stacking ensemble framework to predict borrower default risk in Peer-to-Peer (P2P) lending platforms.

The system integrates Explainable AI (XAI) using SHAP (SHapley Additive exPlanations) to provide both global and individual-level interpretability, ensuring that predictions are transparent, trustworthy, and aligned with financial regulations.

CredixAI transforms predicted probabilities into an intuitive credit score (300–850 scale) and categorizes borrowers into five risk levels, enabling actionable decision-making for financial institutions, fintech platforms, and regulators.

---

## ✨ Key Features

- Hybrid MLP–XGBoost stacking ensemble model  
- High-performance credit risk prediction  
- Explainable AI with SHAP visualization  
- Credit score transformation (300–850 scale)  
- Risk segmentation into 5 categories  
- Interactive web app built with Streamlit
